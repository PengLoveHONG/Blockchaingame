How to Play

Enter a game

Connect to your game session through the game ID and enter your username. Your username will be connected to a private key you can use to sign transactions.

Transaction Period

Before mining begins, every player is able to send transactions to each other. During this period you can purchase perks/bonuses or send money to other players.

Start Mining

Once the transaction period is over, the game will start mining. Input numbers to guess the correct hash for the current block.

Get Rewarded

The player to first guess the correct hash will verify with the rest of the network. If the player is correct through a majority vote, they will receive crypto as a reward.
